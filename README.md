##  **RP Dice**

RP Dice is a plugin adding a set of configurable dices for minecraft roleplay. Server owners can either set presets for dices or allow players to use custom dices.
Config has a configurable message with placeholders (also supports PAPI placeholders). There is also configurable command cooldown to stop command spamming.

This plugin was made as an addition for [Character Cards](https://modrinth.com/plugin/character-cards)

### Commands
```
/rpdice help | /rpdice help message
/rpdice reload - reload plugin (requires rpdice.reload permission) [to load dice presets requires full restart]
/roll <name> - roll a dice preset in the config (requires rpdice.roll)
/roll custom <dice> - roll custom diece (e.g. 1d6, 1d20, 3d4) (requires rpdice.roll.custom)
/roll action <action> - rolls result for specific action (instead of number result, you get text results defined in config)
```

### Permissions
```
rpdice (lack of it blocks access to /rpdice command)
rpdice.reload
rpdice.roll
rpdice.roll.custom
rpdice.roll.action
rpdice.updateinfo (notifications about newer versions)
```
